## ğŸš€ How Can I Contribute?

### 1. Reporting Bugs
* Check the [Issues Tab](https://github.com/mansi066/careerhub/issues) to see if the bug has already been reported.
* If not, open a new issue. Clearly describe the bug, the steps to reproduce it, and the expected behavior.

### 2. Suggesting Enhancements
* We are always looking for new features (AI recommendations, NextAuth, etc.).
* Open an issue with the tag `enhancement` to discuss your idea.

### 3. Pull Requests (PRs)
* **Step 1:** Fork the repository.
* **Step 2:** Clone your fork locally:
  
```bash
git clone [https://github.com/mansi066/careerhub.git](https://github.com/mansi066/careerhub.git)
```
* **Step 3:** Create a feature branch:

```
git checkout -b feature/your-amazing-feature
```
* **Step 4:** Install dependencies and make your changes:

```
npm install
npm run dev
```
* **Step 5:** Commit your changes with a clear message:

```
git commit -m "feat: added dark mode toggle to dashboard"
```
* **Step 6:** Push to your branch and open a Pull Request.
